## Startup

1. Install dependencies

```bash
pnpm install
```

2. Run the app.

```bash
pnpm run dev
```

Wait for bundling.

3. Open the chrome://extensions/ (check the DEVELOPER MODE, we need to `Load unpacked` to the cloned path).
